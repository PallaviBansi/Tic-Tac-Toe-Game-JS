# Tic-Tac-Toe-Game-JS
Tic Tac Toe Game
A classic Tic Tac Toe game implemented in JavaScript. Play against a friend or test your skills against an AI. This project demonstrates basic game logic, user interface design.

Features:
Two-player mode.
